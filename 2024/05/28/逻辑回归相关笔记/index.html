# 二。计算逻辑回归模型的准确率

​ 逻辑回归模型的准确率通常使用混淆矩阵 (confusion matrix) 来计算。

准确率=正确分类的记录数总记录数准确率=\frac{正确分类的记录数}{总记录数} 准确率=总记录数正确分类的记录数​

准确率=cm[0,0]+cm[1,1]cm[0,0]+cm[0,1]+cm[1,0]+cm[1,1]准确率=\frac{cm[0,0]+cm[1,1]}{cm[0,0]+cm[0,1]+cm[1,0]+cm[1,1]} 准确率=cm[0,0]+cm[0,1]+cm[1,0]+cm[1,1]cm[0,0]+cm[1,1]​

​ 即混淆矩阵中对角线上的元素为正确记录数。
